# Feature Requests

Capture capabilities the user wanted but the current workflow, tooling, or skill set could not provide yet.

**Areas**: frontend | backend | infra | tests | docs | config | general
**Statuses**: pending | in_progress | resolved | wont_fix

## Entry Template

```markdown
## [FEAT-YYYYMMDD-001] capability-name

**Logged**: 2026-03-24T09:00:00Z
**Priority**: medium
**Status**: pending
**Area**: general

### Requested Capability
What the user wanted to do

### User Context
Why they needed it and what was blocked

### Complexity Estimate
simple | medium | complex

### Suggested Implementation
How the capability could be added or what it should extend

### Metadata
- Frequency: first_time | recurring
- Related Features: existing-feature-name

---
```
